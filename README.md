## REQUIREMENTS
You must have Python installed unto your computer before accessing.

## Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/sprivera-pshs/IQ-Rush.git
    ```
2.  **Navigate to the game directory:**
    ```bash
    cd IQ-Rush
    ```
3.  **Run the Python script for that game**
    ```bash
    python Iq Rush.py
    ```
